⚜️ Colors of Deception
A Game of Cards & Hidden Shadows
A dark fantasy card battler blending UNO-style card play with hidden social deduction roles. Outsmart your opponents, expose the Shadow Court — or rule from the shadows yourself.

📖 Game Description
Colors of Deception is a browser-based card game built with vanilla JavaScript, HTML, and CSS.. Players are secretly assigned roles, some loyal to the realm, others servants of darkness. Using a color-and-number matching card system (inspired by UNO and AVALON), each player must balance playing cards strategically while using their unique one-time abilities to outmaneuver the opposing faction.
The game supports 1 to 4 AI opponents, all of which think and act autonomously. Roles are hidden — trust no one.
🎮 How to Play
🃏 Card Rules
Cards have a number (1–10) and a color (Red, Blue, Green, Yellow)
On your turn: play a card matching the color OR number of the top discard card
If you cannot play, draw one card from the deck (1 draw per turn maximum)
☠️ Curse Cards — Drawing one causes your next turn to be skipped automatically
First player to empty their hand wins for the Good team — or eliminate all enemies

🚀 Setup & How to Run
No installations, no dependencies. Just a browser.
Option 1 — Open Directly
1. Download or clone this repository
2. Open index.html in any modern web browser
3. That's it — the game runs immediately

Option 2 — Clone via Git
git clone https://github.com/YOUR-USERNAME/colors-of-deception.git
cd colors-of-deception
# Open index.html in your browser

Option 3 — Live Server (Recommended for Development)
If you have VS Code with the Live Server extension:
1. Open the project folder in VS Code
2. Right-click index.html → "Open with Live Server"
